# UART

UART (Universal Asynchronous Receiver-Transmitter) is a hardware communication protocol 
used to communicate between devices. Due to no shared clock, the communication is asynchronous. Instead, both
devices agree on a common baud rate(same pace) to synchronize the communication.

There are two wires:
- TX (Transmit) - sends data
- RX (Receive) - receives data


Characteristics:
- Asynchronous

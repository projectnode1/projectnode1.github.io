# One-Wire

One-Wire is a communication protocol that uses a single wire for both data and 
power transmission. It works by each device having a unique address as a factory
default. This allows for multiple devices to be connected to the same bus and
addressed individually. Master sits at high voltage and devices pull 
the line low to communicate.

Pros:
- Single wire for both data and power transmission, reducing the number of connections needed
- Support for multiple devices on the same bus

Cons:
- Limited transmission distance
- Slower compared to other protocols
